## Features
* Top Chart Songs & Top Artists from Shazam's API
* Top Songs In Your Country (Geolocation API)
* Song Details With Lyrics & Related Songs
* Search Bar
* Responsive Design
* Custom Built Music Player
* Auto Playing The Next Song
